PROJECT LIFELINE

1. About this Web Demonstration

A front-end browser demonstration built to showcase the core state machine, entity triage, and dispatching logic under strict hackathon portability constraints (runs on a single double-click, zero backend setup, serverless execution). It connects browser voice inputs directly to the Google Gemini 3.1 Flash Lite API for extraction/NLP, uses ElevenLabs v2.5 Turbo to solve latency, and cascades to Web Speech synthesis APIs for seamless, low-latency client-side interaction.

2. Real-World Production Deployment

Designed to be deployed as a completely secure, offline system inside municipal emergency communication centers (ECCs). It operates on on-premise local server hardware hosting quantized local models (such as Gemma-9B) alongside offline STT engines (like Whisper) and local TTS voices. This architecture ensures absolute compliance with public security regulations and privacy laws, preventing sensitive emergency transcripts, voice streams, and caller coordinates from ever leaving the internal government intranet.

Cognitive Processing Pipeline (Logic Canvas)
1

VAD & STT: Captures the caller's voice stream and runs Voice Activity Detection offline.

2

NLP Parse: Performs Natural Language Processing to extract slots, intentions, and key incident entities.

3

GIS Map: Queries geographical indexes to pin resolved addresses and landmarks in real time.

4

FSM Loop: Feeds extracted slots to the state machine to direct dialogue flow and prompt next steps.

5

Severity Triage: Dynamically triages the emergency severity class (Low, Medium, or High).

6

SOP Guidance: Fetches local emergency Standard Operating Procedures to advise dispatcher operations.

7

HIL Handoff: Monitors caller dialog state and escalates safely to a Human-in-the-Loop when triggered.

8

Report Gen: Automatically compiles and writes official incident reports in Markdown upon hangup.

How to Test the Voice-to-Voice Operator
1

Locate the simulated smartphone emulator on the right pane of the dashboard.

2

Click the green Call Button and approve browser microphone permissions.

3
Speak in Vietnamese to report a simulated crisis:
"Có một đám cháy lớn tại ngã tư Nguyễn Đình Chiểu với Cách Mạng Tháng Tám, có người ngạt khói!"
4

Observe the Leaflet map center, the incident badges update, and listen to the AI operator's safety advice.

5

Click the red Hang Up Button to end the call and see the automated incident report compile.
